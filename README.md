# Vision Transformer From Scratch

Implementation of the Vision Transformer (ViT) architecture from scratch using PyTorch.

---

## 🚀 Overview

Vision Transformer (ViT) applies transformer architectures to computer vision tasks by splitting images into patches and processing them as token sequences.

This project focuses on understanding the internal working of Vision Transformers through manual implementation of core components using PyTorch.

---

## 🧠 Architecture Flow

Input Image
    ↓
Image Patching
    ↓
Patch Embedding
    ↓
Positional Encoding
    ↓
Transformer Encoder Blocks
    ↓
MLP Head
    ↓
Classification Output
