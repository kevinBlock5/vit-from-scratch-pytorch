# Vision Transformer From Scratch

Implementation of the Vision Transformer (ViT) architecture using PyTorch.

---

## Features

- Image patching
- Patch embeddings
- Positional embeddings
- Multi-head self-attention
- Transformer encoder blocks
- Classification head

---

## Tech Stack

- Python
- PyTorch

---

## Goal

The purpose of this project is to deeply understand the internal working of Vision Transformers by implementing the architecture from scratch.
